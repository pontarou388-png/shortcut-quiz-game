# ⌨️ ショートカットキー クイズ

Windowsのショートカットキーを楽しく覚えるためのクイズゲームです。
かわいいキャラクターたちが正解・不正解に合わせて応援してくれます。

## 🎮 遊び方

1. 「スタート！」ボタンを押す
2. 4つの選択肢から答えを選ぶ（マウスクリック または キーボードの `1`〜`4`）
3. 「つぎへ ▶」で次の問題へ（`Enter` キーでもOK）
4. 20問終わると結果発表！

## 📋 ルール

- 毎回ランダムに **20問** 出題（出題順・選択肢の並びも毎回変わります）
- 1問5点の **100点満点**
- 3問連続で正解すると、ペンギンが「やったね！」と応援に来てくれます
- 🎵 BGMと効果音つき（右上のボタンでオン・オフ切り替え）
- 収録ショートカットは約55種類（Windows基本操作 + ブラウザ操作）

## 🚀 動かし方

ビルドやインストールは不要です。`index.html` をブラウザで開くだけで遊べます。

### GitHub Pages で公開する場合

1. このリポジトリを GitHub にプッシュ
2. リポジトリの **Settings → Pages** を開く
3. Source を「Deploy from a branch」、Branch を `main` / `(root)` に設定して保存
4. 表示されたURLにアクセス

## 🛠 使用技術

- HTML / CSS / JavaScript（外部ライブラリ不使用）
- スマートフォンでも遊べるレスポンシブ対応
